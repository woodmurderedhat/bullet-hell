# Game Design Document (GDD)

## High Concept
Abstract geometric minimalist bullet-hell roguelite with modular RPG systems.
Short runs (20–35 minutes). Skill-first design. Deterministic systems.

## Core Pillars
- Clarity in Chaos
- Deterministic Systems
- Modular Build Depth
- Performance First (Raspberry Pi 5 target)
- Elegant Minimalism

## Core Loop
1. Enter arena
2. Survive wave
3. Choose upgrade
4. Repeat
5. Boss encounter
6. Meta rewards
